# RunMate React Native 실시간 지도 반영 체크리스트

## 1) 사전 준비
- [ ] 백엔드 실행 확인 (`/v1` API + `/ws` 엔드포인트)
- [ ] 앱 로그인 완료 (토큰 저장됨)
- [ ] 위치 권한 허용
- [ ] Android SDK/adb 설정 완료 (실기기/에뮬레이터 실행 가능)

## 2) 단일 계정 검증 (내 경로 실시간)
1. 홈 -> `달리기 시작`
2. 러닝 시작 버튼 탭
3. 20~30초 이동

기대 결과:
- [ ] 지도에 현재 위치 표시
- [ ] 내 경로 Polyline이 실시간으로 이어짐
- [ ] 거리/페이스 값이 증가
- [ ] 러닝 종료 시 화면 이탈/종료 후 오류 없음

## 3) 2계정 검증 (친구 경로 실시간)
### A 계정(호스트)
1. 러닝 시작
2. 계속 이동하며 좌표 전송

### B 계정(참여자)
1. 홈에서 `같이 달려` 초대 수락
2. 러닝 화면 진입 (`joinMode`)

기대 결과:
- [ ] 상단 `함께 달리는 중` 배지 표시
- [ ] 친구 위치 Marker 표시
- [ ] 친구 경로 Polyline이 2~3초 내 갱신
- [ ] 내 경로와 친구 경로가 동시에 보임

## 4) 예외 케이스
- [ ] 토큰 없음: WS 연결 실패해도 앱 크래시 없음
- [ ] 네트워크 일시 끊김: 지도/내 경로 로컬 동작 유지
- [ ] 러닝 종료/화면 종료: 소켓 누수 없이 정리

## 5) 회귀 체크
- [ ] 기존 단독 러닝 기록 저장 동작 정상
- [ ] 같이달려 초대 생성/수락 API 정상
- [ ] 홈에서 친구 리스트/초대 배너 표시 정상
